# Library App

A simple library management application built as part of **The Odin Project JavaScript course**.

## Features

* Add new books using a form dialog
* Display books as individual cards
* Track whether a book has been read
* Toggle read/unread status
* Remove books from the library
* Responsive grid-based layout
* Dark themed UI with book-inspired card design

## Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)

## Concepts Practiced

* JavaScript constructors and prototypes
* Arrays and objects
* DOM creation and manipulation
* Event listeners
* Forms and dialogs
* Data attributes
* Dynamic UI updates

## How to Run

1. Clone this repository.
2. Open `index.html` in your browser.
3. Start adding books to your library.
